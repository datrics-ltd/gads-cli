### 2026-03-18 — Phase 1.3 Authentication — OAuth2

  - `OAuthTokenSource` — implements TokenSource, auto-refreshes on expiry via stored refresh token

  - `status`: shows developer token, client_id, and login state; detects GADS_ACCESS_TOKEN env var
  - `logout`: calls auth.Logout(), confirms deletion
  - `refresh`: calls auth.ForceRefresh(), shows confirmation (debug token preview with --verbose)
- Dependencies added: `golang.org/x/oauth2`, `golang.org/x/oauth2/google`, `github.com/pkg/browser`

### 2026-03-18 — Phase 3.1 Install Script (install.sh)
- `install.sh` — NEW: Unix install script
  - Detects OS (`uname -s`: linux, darwin, windows/MINGW) and architecture (`uname -m`: amd64, arm64)
  - Maps to binary filename: `gads-{os}-{arch}` (e.g. `gads-darwin-arm64`, `gads-linux-amd64`)
  - Fetches latest release tag from GitHub API (`/repos/{owner}/{repo}/releases/latest`) — no jq required, pure sed parsing
  - Downloads binary via curl or wget (whichever is available)
  - Downloads `checksums.txt` and verifies SHA256 via `sha256sum` or `shasum -a 256`; gracefully skips if neither tool nor checksum file present
  - Installs to `~/.local/bin/gads` (non-root) or `/usr/local/bin/gads` (root); overridable via `GADS_INSTALL_DIR` env var
  - Warns if install dir is not in PATH and prints the export line to add
  - Prints next-steps guide (config set, auth login)
  - Script marked executable (`chmod +x install.sh`)

### 2026-03-18 — Phase 3.1 Install Script (install.ps1)
- `install.ps1` — NEW: PowerShell Windows install script
  - Detects that platform is `windows/amd64` (only supported Windows arch)
  - Fetches latest release tag from GitHub API via `Invoke-RestMethod`; supports `$env:GITHUB_TOKEN` for private repo auth
  - Downloads binary (`gads-windows-amd64.exe`) via `Invoke-WebRequest`
  - Downloads `checksums.txt` and verifies SHA256 via `Get-FileHash`; gracefully skips if not present
  - Installs to `$env:LOCALAPPDATA\Programs\gads\gads.exe` by default; overridable via `$env:GADS_INSTALL_DIR` or `-InstallDir` param
  - Automatically adds install dir to the user's `PATH` env var if not already present (persists via `[Environment]::SetEnvironmentVariable`)
  - Cleans up temp directory in `finally` block
  - Prints next-steps guide (config set, auth login)

### 2026-03-18 — Phase 3.3 Self-Update
- `cmd/update.go` — NEW: `gads update` command
  - Fetches latest release tag from GitHub API (`/repos/{owner}/{repo}/releases/latest`)
  - Compares current version (from rootCmd.Version) against latest tag; exits early if already up to date
  - Detects platform binary name from `runtime.GOOS` + `runtime.GOARCH` (e.g. `gads-linux-amd64`)
  - Downloads binary via asset API URL (with `Authorization: token` + `Accept: application/octet-stream` when `GITHUB_TOKEN` is set) or `browser_download_url` for public repos
  - Verifies SHA256 checksum against `checksums.txt` release asset
  - Atomically replaces the running binary: writes to a temp file in the same directory, then `os.Rename()` (preserves original file permissions)
